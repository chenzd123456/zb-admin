# 项目初始化教程

## 1. 环境准备

### 1.1 安装Node.js
```bash
# 推荐使用nvm管理Node版本
nvm install 16
nvm use 16
```

### 1.2 安装包管理工具
```bash
# 选择npm/pnpm/yarn
npm install -g pnpm
```

## 2. 创建项目

### 2.1 使用Vite创建项目
```bash
pnpm create vite@latest zb-admin --template vue-ts
cd zb-admin
```

### 2.2 安装基础依赖
```bash
pnpm install vue-router@4 pinia axios element-plus
pnpm install sass @types/node --save-dev
```

## 3. 目录结构设计

### 3.1 创建基础目录
```bash
mkdir -p src/{api,assets,components,router,store,utils,views}
```

### 3.2 配置文件
```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import path from 'path'

export default defineConfig({
  plugins: [vue()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, 'src')
    }
  }
})
```

## 4. 基础配置

### 4.1 全局样式
```scss
// src/styles/variables.scss
$primary-color: #409EFF;
$success-color: #67C23A;
```

### 4.2 环境变量
```env
# .env.development
VITE_API_BASE_URL=/api
VITE_APP_TITLE=zb-admin-dev
```

## 5. 启动项目
```bash
pnpm dev
